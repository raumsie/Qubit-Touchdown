# Qubit-Touchdown
A version of the Qubit Touchdown board game using PennyLane to simulate quantum states. 

The idea of Qubit Touchdown comes from Thomas G. Wong, and is referenced in his textbook:

Introduction to Classical and Quantum Computing
https://www.thomaswong.net/introduction-to-classical-and-quantum-computing-1e4p.pdf

Qubit Touchdown
https://www.thegamecrafter.com/games/qubit-touchdown
